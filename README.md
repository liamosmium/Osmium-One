- Osmium One
- Desktop hybrid metal 3D printer by Osmium Labs

Combines micro MIG metal deposition with automatic CNC spindle tool changing in one machine. No sintering furnace required. Build volume: 100x100mm.

- Status
V0.1 — Kinematic tool changer mechanism. Designed in Onshape, printed and tested. Zero play on first attempt.

- Roadmap
- V0.1 — Tool changer proof of concept (current)
- V0.2 — Full motion system with ESP32 control
- V0.3 — Spindle integration
- V0.4 — MIG metal deposition
